# majtak.com

This repository is used for technical communication related to Majtak.

It is intended for:
- Reporting security vulnerabilities
- Discussing technical issues
- Sharing suggestions or improvement ideas

This is **not** the main source code of the platform.

---

## Security Reports

If you find a security issue, report it responsibly.
Do not abuse or exploit any vulnerability.

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact

Serious reports will be reviewed and handled properly.

---

## Issues & Discussions

Use Issues or Discussions for:
- Bugs or unexpected behavior
- Technical questions
- Platform-related suggestions

Please keep reports clear and focused.

---

## Guidelines

- Be respectful
- One issue per report
- Do not post sensitive data

Content that does not follow these rules may be closed.

---

## Contact

For sensitive matters, use official Majtak contact channels.
